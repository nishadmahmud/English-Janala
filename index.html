2. The difference between map(), forEach(), and filter()?

map()

Creates a new array by applying a function to each element

// Double each number
const numbers = [1, 2, 3];
const doubled = numbers.map(n => n * 2);
// Result: [2, 4, 6]

forEach()

Executes a function for each element (no return value)

// Log each number
const numbers = [1, 2, 3];
numbers.forEach(n => console.log(n));
// Output: 1 2 3

filter()

Creates new array with elements that pass a test

// Get even numbers
const numbers = [1, 2, 3, 4];
const evens = numbers.filter(n => n % 2 === 0);
// Result: [2, 4]

4. How JavaScript Promises work?

Key Methods:

  • .then() - Handles fulfilled state

  • .finally() - Runs regardless of outcome

Example:

5. How closures work in JavaScript?

A closure is a function that remembers its outer variables.

Example:
function outer() {
 let count = 0;
 return function inner() {
  return count++;
 }
}

The inner function maintains access to count even after outer function exits.
